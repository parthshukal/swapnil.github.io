process

Below is the process which I strictly follow before designing any App or Website, whether it be for a small Startup or a big Agency.

Research

Understand user behavior, needs and perspectives to better empathize with the user

Ideation

Information broken into meaningful chunks to simplify navigation and bring in clarity

Mockups

A skeletal framework of the system is established and feedback is
gathered

Design

A lot of iterations, combinations and long talks make up the visual
experience

User Flows

I translate our insights into user flows & wireframes, which set the foundation for relevant and engaging solutions

User Testing

It is then tested on actual users, iterated, refined and flavored with
brilliance
